# Scholar

Bibliographic infrastructure for the DocDo systematic review.

## Contents

| File | Purpose |
|------|---------|
| `bib/references.bib` | Master bibliography for the paper |
| `bib/additional-references.bib` | Supplementary references identified during review |
| `bib/reviewed-papers.bib` | BibTeX entries for all papers included in the systematic review |
| `bib/sources.md` | Notes on key sources and search strategy |

## Usage

Reference entries from `bib/references.bib` in the LaTeX manuscript ([docdo-paper](https://github.com/game-guild/docdo-paper)).

Keep citation keys stable once used in the paper draft.
