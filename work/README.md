# Work

Authored research artifacts (papers, frameworks, dissertations).

## Contents

| Directory | Purpose |
|-----------|---------|
| `projects/` | All research outputs: papers, frameworks, dissertations, thesis |
| `references/` | External knowledge resources (not evidence) |

## Projects at a glance

```text
projects/
├── dissertations/      # Long-form degree requirements
├── frameworks/         # Operationalized theoretical constructs
├── papers/             # Conference and journal submissions
└── thesis/             # Undergraduate/Master's culminating work
```

Each project folder typically has a README, outline, and related-work notes.
