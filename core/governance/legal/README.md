# Legal

License and compliance notes for external data sources.

## Structure

```
legal/
├── release-compliance.md   # Checklist before publishing data
└── sources/                # Per-source terms snapshots
    └── <source-id>/
        ├── notes.md
        └── terms-YYYYMMDD.*
```

Downloaded PDFs are copyrighted and must not be redistributed. Only derived data (statistics, labels, aggregates) can go in `artifacts/data/public/`.
