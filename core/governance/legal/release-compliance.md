# Release Compliance Checklist

Before publishing anything under `artifacts/data/public/`:

- [ ] No copyrighted full-text PDFs included
- [ ] No raw database exports with proprietary fields
- [ ] Only derived/aggregated data (statistics, labels, counts)
- [ ] Dataset card (`dataset_card.md`) included
- [ ] License file (`LICENSE-DATASET`) included
- [ ] Checksums (`checksums.sha256`) included
- [ ] All upstream sources registered in `registry/sources.yaml`
