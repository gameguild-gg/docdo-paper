# Registry

Single source of truth for external sources and dataset releases.

- `sources.yaml` — search databases used (PubMed, arXiv, Semantic Scholar, Unpaywall)
- `datasets.yaml` — published dataset releases
- `source-aliases.yaml` — alias normalization (e.g. "pubmed-central" → "pubmed")
- `schemas/` — data structure definitions
- `id-maps/` — cross-source ID mappings
