# Research Log

This folder contains chronological research activity logs.

## Naming Convention

Files are named by year and month: `YYYY-MM.md`

## Template

Create a new file for each month:

```markdown
# Research Log — YYYY-MM

## Week 1 (YYYY-MM-01 to YYYY-MM-07)

### Activities
- [ ] Activity 1
- [ ] Activity 2

### Notes


## Week 2 (YYYY-MM-08 to YYYY-MM-14)

### Activities

### Notes


## Week 3 (YYYY-MM-15 to YYYY-MM-21)

### Activities

### Notes


## Week 4 (YYYY-MM-22 to YYYY-MM-28)

### Activities

### Notes


## Monthly Summary

### Accomplishments


### Challenges


### Next Month Goals

```

## Usage

- Log daily/weekly research activities
- Track progress on papers and projects
- Note ideas, blockers, and decisions
- Link to relevant files and commits
