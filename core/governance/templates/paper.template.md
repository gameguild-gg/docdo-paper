# Paper Template

Standard structure for research papers in this program.

---

## Front Matter

```markdown
# [Title]

**Status:** Draft / Under Review / Published  
**Type:** Conference Paper / Journal Article / Workshop Paper  
**Target Venue:** [Venue Name]  
**Word Limit:** [if applicable]

---

## Abstract

[150-250 words]

---

## Keywords

keyword1, keyword2, keyword3
```

---

## Body Structure

### 1. Introduction

- Problem statement
- Gap identification
- Contribution summary
- Paper structure overview

### 2. Background / Related Work

- Relevant prior work
- Positioning against existing literature
- Gap this paper addresses

### 3. [Core Contribution]

- Framework / Model / Method
- Formal definitions
- Key components

### 4. [Validation / Application]

- Theoretical validation
- Case studies
- Evaluation approach

### 5. Discussion

- Implications
- Limitations
- Threats to validity

### 6. Conclusion

- Summary of contribution
- Future work

## File Checklist

For each paper, ensure these files exist:

- [ ] `README.md` — Project overview
- [ ] `paper.md` — Draft content
- [ ] `outline.md` — Argument architecture
- [ ] `related-work.md` — Literature positioning
- [ ] `scope.md` — Research boundaries
- [ ] `venues-and-targets.md` — Target venues
- [ ] `threats-to-validity.md` — Self-critique
