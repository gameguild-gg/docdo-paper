# Outline Template

Standard structure for paper outlines.

---

## Purpose

The outline establishes the argument architecture before drafting. It should be:
- **Complete** — All major points identified
- **Logical** — Clear progression of argument
- **Bounded** — Scope is explicit

---

## Template

```markdown
# Outline — [Paper Title]

## 1. Introduction

- Problem: [What issue motivates this work?]
- Claim: [What is the central thesis?]
- Contribution: [What does this paper provide?]

## 2. Background

- [Relevant literature area 1]
- [Relevant literature area 2]
- Limitations of existing approaches

## 3. Conceptual Framework

- Definition of [core concept]
- Relation to [parent framework] if applicable

## 4. Formalization / Method

- [Key component 1]
- [Key component 2]
- [Validation approach]

## 5. Discussion

- Implications
- Limitations

## 6. Conclusion

- Summary
- Future work: [specific next steps]
```

---

## Outline Quality Checklist

- [ ] Each section has clear purpose
- [ ] Argument flows logically
- [ ] Claims are supported by prior sections
- [ ] Scope is bounded (what's NOT included)
- [ ] Contribution is explicit
