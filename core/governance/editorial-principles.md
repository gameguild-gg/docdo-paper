# Editorial Principles

Writing standards for the DocDo systematic review.

## Voice & Style

- Use third-person academic voice
- Use present tense for established facts, past tense for study results
- Prefer active voice when describing methodology

## Terminology

- Use "3D organ segmentation" (not "volumetric organ segmentation") as the primary term
- Use "CT" (not "computed tomography") after first definition
- Use "deep learning" (not "DL") after first definition in each section
- Refer to the review as "this systematic review" (not "our survey")

## Citation Style

- Follow the target venue's citation format
- Keep citation keys stable in `scholar/bib/references.bib`
