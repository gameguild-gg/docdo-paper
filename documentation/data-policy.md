# Data Policy

Only `artifacts/data/public/` can be redistributed. Everything else is private.

**Private** (do not share):
- Downloaded PDFs (`artifacts/data/evidence/pdfs/`) — copyrighted
- Raw database exports (`artifacts/data/evidence/raw/`) — may be under database ToS

**Can be published** (as supplementary material):
- Screening decisions and labels
- Aggregated statistics (paper counts, architecture distributions, Dice scores)
- PRISMA flowchart data

PDFs should be `.gitignore`d. CSVs and annotations are small enough for Git.
